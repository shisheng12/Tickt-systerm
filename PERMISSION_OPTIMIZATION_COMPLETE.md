## 🔄 数据流程

### 创建用户流程
```
1. 管理员访问"用户管理"页面
2. 点击"新增用户"
3. 填写表单（账号、密码、角色等）
4. 系统检查用户名唯一性
5. 创建用户并存储
6. 更新用户列表
```

### 登录验证流程
```
1. 用户访问登录页面
2. 输入账号和密码
3. 提交验证请求
4. 系统检查账号是否存在
5. 验证密码是否正确
6. 检查账户是否启用
7. 加载用户角色和权限
8. 跳转到Dashboard
```

### 角色权限配置流程
```
1. 管理员访问"角色管理"
2. 点击角色的"配置权限"
3. 树形结构展示所有权限
4. 勾选需要授予的权限
5. 保存更新
6. 该角色下所有用户权限生效
```

## 🚀 使用指南

### 管理员操作流程

#### 1. 创建角色
1. 访问"系统配置 → 角色管理"
2. 点击"新增角色"
3. 填写角色名称、描述、权限级别
4. 保存后点击"配置权限"
5. 勾选该角色需要的权限

#### 2. 创建用户
1. 访问"系统配置 → 用户管理"
2. 点击"新增用户"
3. 填写用户信息：
   - 用户名（登录用）
   - 初始密码
   - 姓名
   - 角色（从已创建的角色中选择）
   - 团队、邮箱、电话等
4. 保存

#### 3. 修改用户密码
1. 在用户列表中找到目标用户
2. 点击"改密"按钮
3. 输入新密码并确认
4. 保存
