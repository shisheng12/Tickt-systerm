## 📋 操作步骤（1分钟完成）

### 打开命令行
在项目文件夹打开命令提示符或 PowerShell

### 执行命令
```bash
cd "C:\Users\admin\Desktop\claude workspace\客服工单系统7.10\ticket-system"
surge dist
```

### 按提示输入

#### 1. Email（邮箱）
```
email: 输入您的邮箱（任意邮箱，如：demo@example.com）
```
**直接输入后按回车**

#### 2. Password（密码）
```
password: 输入密码（至少8位，如：demo12345678）
```
**输入时不显示，输完直接回车**

#### 3. project path（项目路径）
```
project path: (C:\...\dist)
```
**直接按回车**（使用默认）

#### 4. domain（域名）
```
domain: (random-name-1234.surge.sh)
```
**直接按回车**（使用自动生成的域名）

## 🔄 如果遇到问题

### 问题1：命令找不到
**解决**：
```bash
npm install -g surge
```

### 问题2：输入密码没反应
**正常**：输入密码时不显示任何字符，输完直接回车即可

### 问题3：邮箱已被使用
**解决**：换一个邮箱，或者用已有账号的密码登录
